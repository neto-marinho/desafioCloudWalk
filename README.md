# desafioCloudWalk
Desafio: Ler o log do jogo Quake e gerar um relatório com os dados de cada partida. 

Este projeto contempla a leitura do log do jogo Quake retornando os seguintes dados no relatorio:
game: Numero da partida
total_kills: Quantidade de mortes na partida.
players: Jogadores que se conectaram na partida.
Killed: Jogador e quantidade de vezes que o jogador morreu na partida.
Killer: Quantidade de vezes que o jogador matou outro jogador na partida.

Obs: Não foi realizado a subtração de pontos/ mortes do jogador ao ser morto pelo mundo (world).
